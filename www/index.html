Like TinyURL for your car!

HOW IT WORKS

  1. Find an old android phone. Cracked screen? Perfect-o!
  2. Purchase a $5 OBD-II Bluetooth adapter (OPTIONAL)
  3. Install gping.io on your device & hide it somewhere in your vehicle -- anywhere you can power it! Some vehicles have additional cigarette lighter 'power ports' and even USB ports in various places. Be creative!
  4. Monitor your vehicle from your web browser! Upon launch, gping.io will show you your distinctly private-but-shareable monitoring URL.

FEATURES

  1. Displays GPS tracking information (with map)
    • Uses Parallel Location Modules to provide most possible helpful location information in all situations
    • Location Module #1: Uses Network & WiFi based techniques to determine location
    • Location Module #2: Uses GPS satellites to determine location
  2. Shows OBD-II information such as
    • Vehicle VIN,
    • Current speed,
    • engine RPM,
    • battery voltage (V),
    • engine fault codes and other alarms,
    • fuel level (%),
    • distance travelled since last reset (Km),
    • air temperature (C),
    • and more!
  3. Developer API coming soon
    • poll for information, e.g., getFuelLevel() etc.
    • HTTP POST on event alarm, i.e. fuel level drops below 15%.
    • Export map KML / CSV log data

Compatible Bluetooth Adapters

  • ELM327 MiNi OBD or any other 'Prolific' or 'FTDI' chipset device.
    • Reminder: You don't need this piece, but having it will provide more features and abilities.
  • Works on vehicles made by Ford, GM/Vauxhall/Opel, Chrysler, Mercedes, Volkswagen, Audi, Jaguar, Citroen, Peugoet, Skoda, Kia, Mazda, Lexus, Daewoo, Renault, Mitsubishi, Nissan, Honda, Hyundai, BMW, Toyota, Seat, Dodge, Jeep, Pontiac, Subaru and many more.

Usage Tips

  • gping.io is designed to automatically start at boot (along with GPS, OBD, and Network services)
  • In Wireless, Disable WiFi. Parking your car too close to a WiFi signal may cause the signal to flap between WiFi and network data. Disable WiFi to increase reliability.
  • In Location & Privacy, enable both GPS satellites and Network location ("Use wireless networks").
  • Enable Bluetooth if using an OBD adapter, disable bluetooth if not.
  • Enable data & background data.
  • Uncheck Development -> Stay Awake
